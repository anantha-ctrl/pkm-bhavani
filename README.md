# PMK Bhavani Constituency Website

A premium, modern political landing page and administrative dashboard for the PMK Bhavani constituency. This project features a high-end UI/UX, a membership card generation system, and a secure internal dashboard for managing member data and public enquiries.

## 🌟 Key Features

### 🏛️ Public Landing Page (`index.html`)

- **Premium Hero Section**: Dynamic image carousel featuring local leadership and constituency highlights.
- **Leader Spotlight**: Interactive mentor section with automatic image rotation and party-aligned aesthetic frames.
- **Constituency Information**: Dedicated sections for about the party, constituency achievements, and upcoming goals.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

### 💳 Membership Card System (`membership.html`)

- **VIP ID Generation**: Dynamic membership card generator with real-time preview.
- **Validation**: Unique Voter ID verification to prevent duplicate registrations.
- **Downloadable Cards**: High-quality ID card export for members to print or save.

### 🛡️ Admin Intranet Dashboard (`intranet.html`)

- **Secure Access**: Password-protected administrative panel (`pmk2026`).
- **Data Management**: View, search, and filter the complete database of members and petitions.
- **Export Functionality**: One-click CSV export for offline processing of member lists and public queries.
- **System Maintenance**: Built-in tools for data cleanup and system resets.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Styling**: Bootstrap 5, FontAwesome 6, Google Fonts (Outfit & Mukta Malar)
- **Persistence**: LocalStorage (Prototyped for local functionality)
- **Animations**: AOS (Animate on Scroll), Hover.css, Custom CSS Keyframes

## 📁 Project Structure

- `index.html`: Main landing page
- `membership.html`: Membership registration and card generation
- `intranet.html`: Administrative management panel
- `style.css`: Primary stylesheet with custom design tokens
- `membership.css`: Specific styling for ID cards
- `script.js`: Core website logic and form handling
- `membership.js`: ID card generation and validation logic
- `/Images`: Project assets and branding

## 🚀 Getting Started

1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser.
3. To access the admin panel, navigate to "Admin" in the footer or open `intranet.html` directly (Default Password: `******`).
